Offline Voice-Based Science Learning App (Hindi + English)

## Problem Statement
Students in rural areas often struggle to access quality education due to:
- Low or unstable internet connectivity  
- Language barriers (most platforms are English + text-heavy)  
- Difficulty learning through reading-only content  

## Solution
**ShikshaVaani** is an offline-first, voice-based science learning web app designed for rural learners.
It provides:
- Audio-based lessons using Text-to-Speech  
- Video lesson support  
- Voice question answering system  
- Voice-assisted quiz for self-assessment  
- Works as a **PWA (Progressive Web App)** for offline learning
