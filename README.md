The app supports learning through **audio + video lessons**, voice interaction, and simple voice-assisted quizzes.

---

## Problem Statement
Many rural students struggle to access quality education because most online learning platforms:
- require continuous internet access
- depend heavily on reading English content
- are not designed for learners who prefer Hindi or voice-based learning

This leads to learning gaps in foundational subjects like Science.

---

## Solution
ShikshaVaani provides:
- **Hindi + English mode**
- **Audio lessons** using Text-to-Speech (TTS)
- **Video lesson support**
- **Voice-based Question Answering**
- **Voice Quiz** for concept checking
- Works as a **PWA (Progressive Web App)** for offline access
